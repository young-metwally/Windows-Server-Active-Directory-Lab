# Windows Server Active Directory Lab

This project shows a small company network built in VirtualBox.

What is used:

- Windows Server 2022
- Windows 10 Pro
- Active Directory
- DNS
- VirtualBox

## What Is Practiced

- Creating employee accounts
- Resetting passwords
- Managing permissions
- Controlling computer access
- Joining Windows 10 to a domain
- Using Active Directory
